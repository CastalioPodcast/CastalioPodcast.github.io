Episódio 56: Splinter

Date Sun 07 December 2014 Tags splinter / selenium

O que e' o Splinter? O Selenium? Para que serve?

Selenium IDE um add-on para o Firefox que permite gravar e reproduzir interacao com um navegador.

Selenium WebDriver seria uma interface que permite interagir com um navegador usando uma linguagem de programacao.

Usando o Splinter, vamos abrir um navegador Firefox e visitar o site do Castalio no Github, e checar o titulo da pagina:

In [1]: from splinter import Browser

In [2]: navegador = Browser('firefox')

In [3]: navegador.visit('https://castaliopodcast.github.io')

In [4]: print navegador.title
Castálio Podcast

In [5]: navegador.screenshot(name='castalio')
Out[5]: '/var/folders/c3/gg3q87ys51z86n43xgxrh7ch0000gn/T/castaliowfOsTi.png'

In [6]: navegador.quit()

Como voces podem ver, o codigo acima ate mesmo tira um screenshot da pagina que foi aberta no navegador e salva no arquivo de sistemas:

Minha contribuicao

Agora, vamos fazer a mesma coisa, mas desta vez usando o Selenium WebDriver:

In [1]: from selenium import webdriver

In [2]: navegador = webdriver.Firefox()

In [3]: navegador.get('https://castaliopodcast.github.io')

In [4]: print navegador.title
Castálio Podcast

In [5]: navegador.save_screenshot('webdriver-castalio.png')
Out[5]: True

In [6]: navegador.quit()

Quais sao as vantagens em usar o Splinter?

Buscando elementos:

browser.find_by_css('h1')
browser.find_by_xpath('//h1')
browser.find_by_tag('h1')
browser.find_by_name('name')
browser.find_by_id('firstheader')
browser.find_by_value('query')

Os metodos acima retornam uma lista de elementos, se encontrados, o que facilita encontrar elementos por indice. O Splintr tambem adiciona dois metodos (syntatic sugar) que permitem acessar o primeiro e ultimo elemento:

browser.find_by_name('name').first
browser.find_by_name('name').last

Buscando por links:

links_found = browser.find_link_by_text('Link for Example.com')
links_found = browser.find_link_by_partial_text('for Example')
links_found = browser.find_link_by_href('http://example.com')
links_found = browser.find_link_by_partial_href('example')

Encadeando buscas:

divs = browser.find_by_tag("div")
within_elements = divs.first.find_by_name("name")

Quais sao as desvantagens?

Limitacao de interacao com o mouse que somente funcionam nos navegadores Chrome e Firefox 27.0.1
